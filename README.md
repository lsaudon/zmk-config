# ZMK Config - Sofle Choc Pro BT (Keebart)

Configuration ZMK pour le clavier split **Sofle Choc Pro BT** de [Keebart](https://keebart.com).

## Configuration

| Option           | Valeur      |
|------------------|-------------|
| Nom Bluetooth    | `Christian` |
| Émulation souris | activée     |

## Layers

Le keymap comporte 4 layers, activables via les touches **LOWER** et **RAISE** sur les pouces.
LOWER + RAISE active automatiquement la layer **ADJUST** (conditional layer).
